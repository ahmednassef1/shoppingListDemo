# Shopping List Demo

An Android application for managing shopping lists.

## Requirements

- Android Studio
- Android SDK (minimum SDK version TBD)
- Gradle

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ememobile/shoppingListDemo.git
   ```
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Run the app on an emulator or physical device.
